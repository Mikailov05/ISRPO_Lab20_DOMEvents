## Что изучили

### 1. DOM (Document Object Model)
DOM — это объектная модель HTML-документа, представленная в виде древовидной структуры. JavaScript использует DOM для:
- Чтения элементов страницы и их содержимого
- Изменения текста, HTML-разметки и атрибутов элементов
- Управления CSS-стилями элементов
- Реагирования на действия пользователя (события)

### 2. Поиск элементов в DOM
Освоены основные методы поиска элементов:
- `getElementById()` — поиск по уникальному идентификатору
- `querySelector()` — универсальный поиск по CSS-селектору
- `querySelectorAll()` — поиск всех элементов по селектору

### 3. Изменение содержимого и стилей
Изучены способы изменения элементов:
- `textContent` — изменение текстового содержимого
- `style` — изменение CSS-стилей напрямую из JavaScript
- `classList` — управление классами элемента

### 4. Обработка событий
Освоена работа с событиями через `addEventListener()`:
- Событие `click` — обработка кликов
- Событие `submit` — обработка отправки форм
- Событие `input` — реагирование на ввод данных

### 5. Работа с формами и валидация
Изучены методы работы с формами:
- Получение значений из полей ввода через `.value`
- `event.preventDefault()` — отмена стандартной отправки формы
- Валидация данных (проверка на пустоту, диапазон значений)
- `trim()` — удаление лишних пробелов
- `focus()` — установка фокуса на поле ввода
- `reset()` — очистка формы

### 6. Динамическое создание элементов
Освоены методы создания и удаления элементов:
- `createElement()` — создание нового HTML-элемента
- `appendChild()` — добавление элемента в DOM
- `remove()` — удаление элемента

### 7. Делегирование событий
Изучен важный паттерн для работы с динамически созданными элементами:
- Обработчик вешается на родительский элемент
- Событие "всплывает" от дочернего элемента к родителю
- Позволяет обрабатывать события даже для элементов, созданных после загрузки страницы
